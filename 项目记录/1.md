# 1、初始化项目

使用PyCharm初始化项目。

# 2、引入`Psycopg`

Psycopg 3 是为 [Python](https://www.python.org/) 编程语言新设计的 [PostgreSQL](https://www.postgresql.org/) 数据库适配器。

## 1、安装

```python
pip install --upgrade pip           # upgrade pip to at least 20.3
pip install "psycopg[binary]"       # remove [binary] for PyPy
```

由于安装Psycopg至少需要**pip 20.3**，因此需要先将pip更新。

## 2、使用

在项目根目录的`settings.py`中设置DATABASES

```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'testDatabase',
        'USER': 'postgres',
        'PASSWORD': 'postgresql',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```

# 2、引入`GeoServer-rest`

安装本软件包需要先安装以下软件包:

- [GDAL](https://gdal.org/)

在windows中，可以使用`pipwin`来安装`gdal`依赖。

```python
pip install pipwin
pipwin refresh
pipwin install gdal
```

现在可以使用pip install命令安装库

```python
pip install geoserver-rest
```
